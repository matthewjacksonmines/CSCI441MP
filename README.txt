Fall 2025 - MP - The Alchemist's Guild

This program is a simple program with 4 controllable characters. 3 different camera types. Phong reflectance illumination. Skybox texturing

When executing the program, use WASD (or arrow keys) to control movement.
P to switch characters
C to change between arcball and freecam
 - Freecam controls:
    - SPACE / CTRL + SPACE to move forward/backward
    - LMB + Move Mouse to rotate camera
V to toggle first-person picture-in-picture viewport
SPACE / CTRL + SPACE to zoom the camera in/out
ESC or Q to close program.
R to reload shaders.

Compile using cmake via CMakeLists.txt. 
Run the executable from the top level directory.
